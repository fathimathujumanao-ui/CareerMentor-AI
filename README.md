# 🚀 CareerMentor AI

An AI-powered Career Guidance Platform built using React, FastAPI, Groq LLM, and Docker.

## 📌 Project Overview

CareerMentor AI helps students and job seekers make informed career decisions using Generative AI. The application provides personalized career recommendations, resume analysis, skill gap identification, interview preparation, and downloadable PDF reports.

---

## ✨ Features

- 🎯 AI Career Recommendation
- 📄 Resume Analysis
- 📊 Skill Gap Analysis
- 💼 Interview Preparation

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/fathimathujumanao-ui/CareerMentor-AI.git
```

### Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
